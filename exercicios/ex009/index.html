Triade do front-end: (Uma das partes fundamentais da WEB)

HTML
HTML é a linguagem de marcação, é o conteúdo e semântica da página.
CSS
CSS é a linguagem de estilo, que cuida de tamanhos, cores, posicionamentos e etc.
JavaScript
JavaScript é a linguagem de programação que vai cuidar da interatividade e dinamicidade da página.

Alguns termos do League of Legends:

Flashar
É o ato de utilizar o feitiço "FLASH" em alguma direção.
Dar TP
É o ato de utilizar o feitiço "Teleporte" em algum lugar.
Cinzou
É quando você morre, pois a sua tela fica cinza.
Feedar
É o ato de se matar ou morrer muito para o adversário e assim deixar ele forte demais.
